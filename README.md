##💼 E-Learning Platform UI Design ✨
The E-Learning Platform UI Design is a modern, responsive, and interactive front-end web application developed to simulate a real-world Learning Management System (LMS). This project is designed to provide a structured digital learning environment where users can explore courses, watch video lectures, access study materials, and track their learning progress in an efficient and user-friendly manner. It demonstrates how educational platforms can be built using core front-end technologies and serves as a strong foundation for understanding real-world LMS applications.

The main goal of this project is to create an engaging learning experience using only front-end technologies without relying on a backend server. It replicates key functionalities of professional e-learning platforms such as course browsing, user login simulation, video playback, and progress tracking. This helps users experience a complete learning workflow within a single web interface.
The application begins with a login interface, where the user enters a username to access the platform. After successful login, the user is redirected to the homepage, which displays a personalized welcome message. This creates a simple session-like experience using browser-based storage. The homepage acts as the central navigation point, allowing users to access different sections such as Courses, Dashboard, Profile, and Logout.

The course listing section displays multiple technical courses including HTML & CSS, JavaScript, Python, and SQL. Each course is presented using interactive cards with hover effects to enhance the user experience. These cards are designed to be visually appealing and clickable, leading users to the course player section.

The course detail or player section is the core feature of the platform. When a user selects a course, they are shown an embedded YouTube video lecture along with a link to external study notes. This allows users to learn directly within the platform without switching applications. An “Enroll” button is provided to simulate course enrollment and track learning progress. The progress data is stored using localStorage so that it remains saved even after refreshing the browser.

The progress tracking dashboard displays the completion status of enrolled courses. It retrieves stored data from localStorage and presents it in an organized format. This feature demonstrates how user activity can be managed on the client side without using a backend database. It also introduces basic concepts of state management in front-end development.
The profile section displays basic user information such as the username entered during login. This simulates a simple user account system. The logout functionality clears all stored session data and redirects the user back to the login page, ensuring proper session handling.

🛠️ Tools & Technologies Used
The E-Learning Platform UI is built using modern front-end technologies:

HTML5: Used to structure the application and define sections such as login, home, courses, dashboard, and profile.

CSS3: Used for styling and layout design. It enhances visual appearance using Flexbox, Grid, gradients, shadows, hover effects, and responsive design techniques to ensure compatibility across devices.

JavaScript (Vanilla JS): Used to add interactivity, handle navigation, manage user login simulation, course selection, video embedding, and progress tracking using localStorage.

YouTube Embedded Player: Used to display video lectures directly within the application for seamless learning.
External Notes Links: Used to provide access to study materials and reference documents.

LocalStorage API: Used to store user progress and login data in the browser, enabling persistence without a backend system.

Git & GitHub: Used for version control and project hosting, allowing easy sharing and collaboration.

⚙️ How It Works
The application is built as a single-page interactive system. HTML defines the structure, CSS handles styling and responsiveness, and JavaScript manages functionality. LocalStorage is used to store user data and progress. When a user logs in, their username is saved and displayed across the platform. As they enroll in courses, progress is updated and reflected in the dashboard.
